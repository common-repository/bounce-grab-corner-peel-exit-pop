== Description ==

Bounce Grab is an easy to use Wordpress plugin that shows a corner peel advertisement when a user tries to exit your page.

Allowing you to show multiple to people when they try and leave your site.

== Installation ==

1.  Upload the entire bunce grab folder to your wp-content/plugins/ directory.  
  
2.  Activate the plugin through the \'Plugins\' menu in WordPress.  
  
3.  Use the \'Bounce Grab\' menu under the settings section to configure your settings
